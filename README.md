# Projeto-Nikel_-GrowDev2025
Projeto desenvolvido para prática das tecnologias **HTML**, **CSS** e **JavaScript**, seguindo os princípios de estruturação limpa, código organizado, responsividade e interatividade.   O objetivo é apresentar uma página completa com apresentação do projeto, serviços, galeria, formulário de contato e recursos básicos de interação.
